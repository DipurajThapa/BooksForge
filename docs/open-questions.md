# Open Questions — BooksForge

Append questions here when a spec is ambiguous, contradictory, or silent on something that must be decided. Per `CLAUDE_CODE_START_HERE.md §7`: make a defensible best-guess, mark it `[ASSUMED]` in code, surface it in the PR, and continue. Do not block on questions.

Format:

```
## [ASKED-YYYY-MM-DD] Short question title

**Context:** Why this came up.
**Options:** A, B, (or more).
**Assumed:** Option X — one-line rationale.
**Resolved:** YYYY-MM-DD — what was decided and by whom. (Delete this line until resolved.)
```
